# Changelog

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-17

### Added
- `WithSuccessSampling` middleware option to allow dropping logs for successful requests.

### Changed
- Renamed basic example from `main.go` to `server.go`.
- Simplified README for better clarity and usability.
- Fixed outdated references in documentation.

## [0.0.1] - 2026-01-05

### Added
- Initial implementation of context-based field accumulation
- HTTP middleware for request logging with functional options
- GitHub Actions CI workflow
- Comprehensive test suite
- Documentation and examples
